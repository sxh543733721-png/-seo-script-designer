✨ 3分钟上手教程
────────────────────────────────────

📌 第一步：打开工具
  1. 找到「index.html」文件
  2. 双击打开（会在浏览器中打开）
  3. 看到紫色渐变界面就成功了！

📌 第二步：配置参数（推荐新手配置）
  
  基础设置：
  ✓ 目标受众：选择「中小企业主」
  ✓ 视频时长：选择「2-3分钟」
  
  风格定制：
  ✓ 内容风格：选择「问题解决型」
  ✓ 发布平台：选择「YouTube」
  
  痛点选择（勾选2-3个）：
  ✓ 外包太贵 ✓
  ✓ 学习太难 ✓
  ✓ 不会设计 ✓
  
  功能侧重（勾选3-4个）：
  ✓ AI对话建站 ✓
  ✓ 60秒生成 ✓
  ✓ 实时修改 ✓
  ✓ 移动适配 ✓
  
  行动召唤：
  ✓ 选择「注册试用型」
  
  语言版本：
  ✓ 中文简体 ✓ (默认)
  ✓ English ✓ (可勾选)
  
  SEO优化级别：
  ✓ 选择「专业级」

📌 第三步：生成脚本
  1. 点击底部的「✨ 生成脚本」按钮
  2. 等待1秒，页面自动滚动到结果区域
  3. 看到两个文本框（中文+英文）

📌 第四步：复制使用
  1. 点击「📋 复制」按钮
  2. 粘贴到配音软件（讯飞、剪映等）
  3. 开始录制视频！

💡 使用技巧
────────────────────────────────────

技巧1：多试几次
  • 调整不同参数组合
  • 生成多个版本对比
  • 找到最适合你的风格

技巧2：根据平台调整
  • YouTube：选「专业级」SEO
  • 抖音：选「30-60秒」时长
  • B站：选「3-5分钟」深度内容

技巧3：痛点不要太多
  • 短视频：2个痛点就够
  • 中视频：3个痛点合适
  • 长视频：可以全选

技巧4：功能要精选
  • 核心功能：AI对话、60秒生成
  • 附加功能：根据时长选择
  • 不要全选（除非长视频）

技巧5：SEO优化
  • YouTube必选「专业级」
  • 会自动生成标题、标签、时间戳
  • 直接复制到视频描述

🐛 常见问题
────────────────────────────────────

Q: 打开index.html没反应？
A: 确保用浏览器打开（Chrome/Safari/Edge）
   如果还不行，用本地服务器方式打开

Q: 生成的脚本太长/太短？
A: 调整「视频时长」参数
   短视频少选痛点和功能
   长视频多选痛点和功能

Q: 想改变语气风格？
A: 调整「内容风格」参数
   专业严谨：正式场合
   轻松幽默：年轻受众
   激情演讲：培训视频

Q: 英文脚本不需要？
A: 取消勾选「English」即可
   只会生成中文版本

Q: 可以保存历史记录吗？
A: 当前版本暂不支持
   建议复制到文档保存
   未来版本会添加此功能

Q: 生成的脚本可以修改吗？
A: 当然可以！
   复制后根据需要调整
   工具只是提供基础框架
